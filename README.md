# BraniacLMS

Современная платформа для обучения. прогрессивный взгляд на простые вещи.
Учебный проект системы управления обучением, который разработан при прохождений обучения на факультете "pyton"

## Стек

- Python > 3.7
  - isort, black, autoflake
  - Django < 3.3
  - Celery[Redis]
- VSCode
- SQLite 3
